**Customer Behavior Analysis Project**


**Overview**

This project demonstrates a complete data analytics workflow, from data loading and cleaning to analysis, visualization, and reporting. The focus is on analyzing customer behavior to extract actionable insights for business decision-making.

**Dataset**

The dataset used in this project contains customer transactions, demographic information, and purchase details. It provides the foundation for exploratory data analysis (EDA), SQL querying, and dashboard creation.

**Tools**

**Python** – Data loading, cleaning, and exploratory analysis

**MySQL Server** – Running SQL queries and data manipulation

**Power BI** – Interactive dashboard creation

**Gamma** – Presentation and reporting



**Steps:**

**Data Loading** – Import dataset into Python for initial exploration.

**Exploratory Data Analysis (EDA)** – Identify trends, distributions, and anomalies.

**Data Cleaning** – Handle missing values, duplicates, and inconsistencies.

**SQL Queries** – Extract, filter, and aggregate data using MySQL Server.

**Dashboard Creation** – Visualize key metrics in Power BI for insights.

**Reporting** – Summarize findings in a report and create a presentation using Gamma.



**Dashboard:**

The Power BI dashboard highlights:

Revenue and sales trends

Customer segmentation by demographics

Product popularity and purchase patterns

Shipping methods and revenue impact



**Results:**

Identified high-value customer segments and revenue drivers

Highlighted opportunities for targeted marketing and promotions

Provided clear visual insights for decision-making



**How to Run**

1. Clone the repository:

git clone https://github.com/Marc3n/customer_behavior_analysis.git


2. Open the Python scripts and run in your preferred IDE to perform EDA and cleaning.

3. Use the provided SQL scripts to query the dataset in MySQL Server.

4. Open the Power BI dashboard file to explore interactive visualizations.

5. Refer to the Gamma presentation for summarized insights.
